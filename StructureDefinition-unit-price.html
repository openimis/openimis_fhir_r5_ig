Extension: Unit Price (Medication|ActivityDefinition)

Official URL: https://github.com/openimis/openimis_fhir_r5_ig/StructureDefinition/unit-price Version: 0.1.0
Draft as of 2025-08-26 Computable Name: UnitPriceExtension

The unit price for a Medication or ActivityDefinition.

Context of Use

    This extension may be used on the following element(s)

    • Element ID: OpenIMISMedication
    • Element ID: OpenIMISActivityDefinition
    • Element ID: OpenIMISInvoice
    • Element ID: ActivityDefinition
    • Element ID: Invoice.lineItem.priceComponent
    • Element ID: Medication

Usage info

Usages:

  • Use this Extension: openIMIS ActivityDefinition, openIMIS Bill, openIMIS Invoice and openIMIS Medication
  • Examples for this Extension: A1, Invoice/InformalSectorContributionInvoice and Medication/AspirinMedication
