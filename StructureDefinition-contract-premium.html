Documentation for this format

This structure is derived from Extension

Summary

Complex Extension: Indicates a premium for a Contract. This is a complex extension.

  • payer: CodeableConcept: Specifies who paid for the premium.
  • category: CodeableConcept: Specifies the premium category.
  • amount: Money: Specifies hoq much has been paid for the premium.
  • receipt: string: Specifies the premium receipt.
  • date: date: Specifies when the premium has been paid.
  • type: CodeableConcept: Specifies the payment type for the premium.
