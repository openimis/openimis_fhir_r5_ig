Resource Profile: openIMIS Location

Official URL: https://github.com/openimis/openimis_fhir_r5_ig/StructureDefinition/openimis-location Version: 0.1.0
Draft as of 2025-08-26 Computable Name: OpenIMISLocation

Defines a Location for openIMIS which maps to a Location

Usages:

  • Refer to this Profile: Location Reference (Address), openIMIS Enrolment Officer Practitioner Role, openIMIS Insurance Plan and openIMIS Location
  • Examples for this Profile: Jamu, Jamula, Jimara and Upper River Region
  • CapabilityStatements using this Profile: openIMIS Capability Statement
